# Weather • Material 3 Dashboard

Material 3 inspired real-time weather dashboard using OpenWeatherMap API.
Features:
- City search, geolocation
- Current weather, humidity, wind, pressure, sunrise/sunset
- 5-day aggregated forecast
- Light/Dark mode, °C/°F toggle
- Responsive & animated Material 3 style

## Usage
1. Add your OpenWeatherMap API key in `script.js` at `OWM_KEY`.
2. Deploy on GitHub Pages (Settings → Pages → branch: main → root).
